### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户（业主）、物业人员、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 小区信息
5. 车位信息
6. 小区公告
7. 留言板
8. 后台管理
9. 个人中心
10. 我的收藏

**物业人员功能：**
1. 登录
2. 修改密码
3. 个人信息
4. 资产信息管理
5. 缴费通知管理




**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 业主管理
4. 小区信息管理
5. 物业人员管理
6. 资产信息管理
7. 业主车辆管理
8. 业主宠物管理
9. 车位信息管理
10. 购买车位管理
11. 缴费通知管理
12. 留言板管理
13. 系统管理（系统简介、轮播图管理、小区公告、关于我们）
